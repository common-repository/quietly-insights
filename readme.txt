A plugin for embedding the Quietly Insights analytics code on your WordPress blog.

== Description ==

Creating great content starts with knowing what works. Quietly Insights helps content marketers focus on their strategy by measuring, analyzing, and summarizing data into actionable recommendations.

Using everything from reader engagement data (such as whether visitors actually finish reading your stories), to competitor performance data, Quietly Insights tells you which ideas have been effective recently, and predicts which ideas will be effective in the future.

Additionally, Quietly Insights simplifies conversions—once you’ve installed the Quietly Insights this plugin, you can set up goals without touching any code. Quietly Insights will report on these goals, and show you which pieces of content contributed to each goal.

To use this plugin, you’ll need a Quietly Insights account. Sign up free at https://insights.quiet.ly

== Installation ==

1. Download and install the quietly-insights plugin from WordPress dashboard.
2. Activate the plugin from the Plugins menu in the WordPress dashboard.
3. Get your Property ID from the Install page of your property settings in Quietly Insights. https://insights.quiet.ly/app/#/0/install
4. Enter your Property ID and click the “verify” button.

Don't have a Quietly Insights account? Sign up for free at https://insights.quiet.ly

Stuck? More detailed installation instructions can be found here: https://insights.quiet.ly/support/dt_articles/install-quietly-insights-website/
